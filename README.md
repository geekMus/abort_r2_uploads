解决在 Cloudflare R2  存储桶中出现的“正在进行的多部分上传”删除不掉的问题。

执行 `yarn install` 或者 `npm install` 安装依赖，`yarn start` 或 `npm start` 开始运行清理程序。

具体使用请看交互式提示。



觉得交互式输入麻烦的话，可提前配置在 config.json 中，格式如下：

```json
{
    "Bucket": "",
    "Prefix": "",
    "Endpoint": "",
    "AccessKeyId": "",
    "SecretAccessKey": ""
}

```
